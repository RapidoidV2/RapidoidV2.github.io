Rapidoid - The No-bullshit Web Framework(s) for Java!

Rapidoid consists of several de-coupled modules/frameworks which can be used separately or together.


Add rapidoid-main as Maven dependency:
<dependency>
    <groupId>org.rapidoid</groupId>
    <artifactId>rapidoid-main</artifactId>
    <version>5.0.0</version>
</dependency>
1 Web page and RESTful service one-liners (http-fast module)
 Let's create a simple web page with 1 line of code:
On.page("/").gui("Hello world!");

 Let's create a simple REST service with 1 line of code:
On.get("/size").json("msg", msg -> msg.length());

Hello world!

2 Searching through collections (http-fluent module)
 Searching through lists or sets:

List<String> items = U.list("a", "bbbbb", "cc");["a","bbbbb","cc"]ArrayList

Find.firstOf(items).where(s -> s.length() < 3);"a"Optional<String>

Find.allOf(items).where(s -> s.length() > 1);["bbbbb","cc"]ArrayList

 Searching through maps:

Map<Integer, String> nums = U.map(1, "one", 2, "two", 3, "three");{"1":"one","2":"two","3":"three"}LinkedHashMap

Find.lastOf(nums).where((k, v) -> v.length() > 4);{"3":"three"}Optional<Entry>

Find.allOf(nums).where((k, v) -> k > 1);{"2":"two","3":"three"}HashMap

5 Automatic JSON Serialization of Data Structures
TextTools.java
@Controller
public class TextTools {

    @GET
    public Map<String, String> upper(String s) {
        String big = s.toUpperCase();
        return U.map("normal", s, "big", big);
    }

    @POST
    public String[] parts(String text) {
        return text.split("-");
    }

}
GET /upper/abc
{"normal":"abc","big":"ABC"}
POST /parts/just-some-words
["just","some","words"]

16 Dependency injection of singletons
Bar.java
@Controller(url = "/bar")
public class Bar {

    @Inject
    public Foo foo;

    @GET(uri = "/hi")
    public String hello() {
        return foo.msg();
    }

    public String msg() {
        return "Hello from Bar!";
    }

}
Foo.java
@Controller(url = "/foo")
public class Foo {

    @Inject
    public Bar bar;

    private int count;

    @GET(uri = "/hi")
    public String hello() {
        return ++count + ": " + bar.msg();
    }

    public String msg() {
        return "Hello from Foo!";
    }

}
GET /foo/hi
"489: Hello from Bar!"
GET /bar/hi
"Hello from Foo!"
GET /foo/hi
"490: Hello from Bar!"
