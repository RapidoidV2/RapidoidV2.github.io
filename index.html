No-bullshit Web Framework for Java!

Getting started :: Adding Rapidoid to your Java project
Add Rapidoid as Maven dependency to your project:
<dependency>
    <groupId>org.rapidoid</groupId>
    <artifactId>rapidoid-main</artifactId>
    <version>4.1.0</version>
</dependency>
Add Main.java as entry point:
public class Main {
    public static void main(String[] args) {
        Rapidoid.run(args);
    }
}

3 Automatic JSON serialization
TextTools.java
@Web
public class TextTools {

    @GET
    public Map<String, String> upper(String s) {
        String big = s.toUpperCase();
        return U.map("normal", s, "big", big);
    }

    @POST
    public String[] parts(String text) {
        return text.split("-");
    }

}
GET /upper/abc
{"normal":"abc","big":"ABC"}
POST /parts/just-some-words
["just","some","words"]
