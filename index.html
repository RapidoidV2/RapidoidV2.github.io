Rapidoid - The No-bullshit (Web) Frameworks for Java!

Open source (Apache license). Simple. Secure. Extremely fast!

Rapidoid consists of several de-coupled modules/frameworks which can be used separately or together: http-fast, fluent, web, u, and more...


Add rapidoid-web as Maven dependency:
<dependency>
    <groupId>org.rapidoid</groupId>
    <artifactId>rapidoid-web</artifactId>
    <version>5.0.3</version>
</dependency>

3 Sweet & Fluent wrapper API for the Java 8 streams (rapidoid-fluent module)
 Manipulating lists or sets:

List<String> items = New.list("a", "bbbbb", "cc");["a","bbbbb","cc"]ArrayList

Find.allOf(items).where(s -> s.length() > 1);["bbbbb","cc"]ArrayList

Do.map(items).to(s -> s.length());[1,5,2]ArrayList

Do.reduce(items).by((a, b) -> a + ":" + b).orElse("");"a:bbbbb:cc"String

Do.group(items).by(s -> s.length());{"1":["a"],"2":["cc"],"5":["bbbbb"]}HashMap

 Manipulating maps:

Map<Integer, String> nums = New.map(1, "one", 2, "two", 3, "three");{"1":"one","2":"two","3":"three"}LinkedHashMap

Do.map(nums).to((k, v) -> k * 1000, (k, v) -> v.toUpperCase());{"2000":"TWO","3000":"THREE","1000":"ONE"}HashMap

Do.group(nums).by((k, v) -> k % 2 == 0 ? "even" : "odd");{"even":{"2":"two"},"odd":{"1":"one","3":"three"}}HashMap

 interface Flow extends Stream { with extras }:

Flow.of("Hi", "there", "X").map(c -> c + "!").filter(s -> s.length() <= 3).toList();["Hi!","X!"]ArrayList
4 RESTful service from POJO (rapidoid-web module)
TextTools.java
@Controller
public class TextTools {

    @GET
    public Map<String, String> upper(String s) {
        String big = s.toUpperCase();
        return U.map("normal", s, "big", big);
    }

    @POST
    public String[] parts(String text) {
        return text.split("-");
    }

}

5 Full application example (rapidoid-web + rapidoid-fluent modules)
Main.java
public class Main extends GUI {

  public static void main(String[] args) {

    Conf.args(args);

     The application will manage this configuration:

    Map<String, String> cfg = U.map("title", "App", "user", "u1");

     RESTful service for configuration retrieval:

    On.get("/cfg").json(cfg);

    PageMenu menu = PageMenu.from(U.map("Home", "/", "Group", "/group"));

     Wrap every result into a page:

    On.defaultWrap((data, next) -> next.invoke(x -> {
        return page(x).menu(menu).title(cfg.get("title"));
    }));

     Create buttons to filter by starting letter:

    List<Btn> letters = Flow.chars('a', 'z').map(c -> {
        return btn(c).linkTo("/find?p=" + c);
    }).toList();

     Display the configuration entries at the home page:

    On.page("/").gui(req -> {
        KVGrid grid = grid(cfg).keyView(k -> a(k).href("/edit?k=" + k));
        return U.list(letters, grid);
    });

     Group the configuration entries by the starting letter:

    On.page("/group").gui(req -> {
        return Do.map(Do.group(cfg).by((k, v) -> k.charAt(0)))
                 .toList((k, v) -> U.list(h3(k), grid(v)));
    });

     Edit configuration entries:
    
    On.page("/edit").gui("k", k -> {
        Btn ok = btn("Update").onClick(() -> Log.info("Updated!"));
        return edit(cfg, k).buttons(ok, btn("Back").linkTo("/"));
    });
    
     Search configuration entries by prefix:

    On.page("/find").gui("p", p -> {
        return grid(Find.allOf(cfg).where((k, v) -> k.startsWith(p)));
    });
  }
}
