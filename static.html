Serving static files

The default locations for serving static files are "public" and "static". All resources that exist in these folders will be automatically served when requested.

1 Serving static files from the default locations

Main.java
import org.rapidoid.setup.On;
 
public class Main {
 
    public static void main(String[] args) {
        On.get("/").html("Home");
    }
 
}
