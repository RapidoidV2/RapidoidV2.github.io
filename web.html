Rapidoid - The No-bullshit Web Framework for Java!

Open source (Apache license). Simple. Secure. Extremely fast!

Community: rapidoid@googlegroups.com
Rapidoid Web High-level POJO Web Framework
The high-level API is built on top of Rapidoid HTTP server, provides many advanced features for building modern POJO-based web applications.


Add rapidoid-web as Maven dependency:
<dependency>
    <groupId>org.rapidoid</groupId>
    <artifactId>rapidoid-web</artifactId>
    <version>5.0.5</version>
</dependency>

3 Automatic JSON Serialization of Data Structures
TextTools.java
@Controller
public class TextTools {
 
    @GET
    public Map<String, String> upper(String s) {
        String big = s.toUpperCase();
        return U.map("normal", s, "big", big);
    }
 
    @POST
    public String[] parts(String text) {
        return text.split("-");
    }
 
}

{"normal":"abc","big":"ABC"}
POST /parts/just-some-words

14 Dependency injection of singletons
Bar.java
@Controller(url = "/bar")
public class Bar {
 
    @Inject
    public Foo foo;
 
    @GET(uri = "/hi")
    public String hello() {
        return foo.msg();
    }
 
    public String msg() {
        return "Hello from Bar!";
    }
 
}
Foo.java
@Controller(url = "/foo")
public class Foo {
 
    @Inject
    public Bar bar;
 
    private int count;
 
    @GET(uri = "/hi")
    public String hello() {
        return ++count + ": " + bar.msg();
    }
 
    public String msg() {
        return "Hello from Foo!";
    }
 
}

17 Server Configuration

Usage:

java -cp <yourapp>.jar com.yourapp.Main [option1 option2 ...]

Example:

java -cp <yourapp>.jar com.yourapp.Main port=9090 address=127.0.0.1 cpus=2 workers=4 nodelay

Configuration options:

Key Value
mode=(dev|production) configure dev or production mode (default: production unless debug mode detected)
secret=<SECRET> configure app-specific secret token for encryption (default: null)
port=<P> listen at port P (default: 8888)
address=<ADDR> listen at address ADDR (default: 0.0.0.0)
threads=<T> start T threads for the job executor service (default: 100)
cpus=<C> optimize for C number of CPUs (default: the actual number of the CPUs)
workers=<W> start W number of I/O workers (default: the configured number of CPUs - cpus options)
nodelay set the TCP_NODELAY flag to disable Nagle's algorithm (default: false)
blockingAccept accept connection in BLOCKING mode (default: false)
bufSizeKB=<SIZE> TCP socket buffer size in KB (default: 16)
