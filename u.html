Rapidoid - The No-bullshit Web Framework(s) for Java!

rapidoid-u Simple & Sweet Utils
The must-have utilities for every Java project! Simple and straight to the point!


Add rapidoid-u as Maven dependency:
<dependency>
    <groupId>org.rapidoid</groupId>
    <artifactId>rapidoid-u</artifactId>
    <version>5.0.1</version>
</dependency>
1 Creating collections
 Creating a list:

List<Integer> numbers = U.list(10, 20, 70);[10,20,70]ArrayList

 Creating a set:

Set<String> words = U.set("abc", "xy", "rapidoid");["abc","xy","rapidoid"]LinkedHashSet

 Creating a map:

Map<Integer, String> nums = U.map(1, "one", 5, "five");{"1":"one","5":"five"}LinkedHashMap
2 Collection utils
List<Integer> words = U.list(10, 20, 30);[10,20,30]ArrayList

Map<Integer, String> nums = U.map(1, "one", 5, "five");{"1":"one","5":"five"}LinkedHashMap

 U.isEmpty utils:

U.isEmpty(words);falseBoolean

U.isEmpty(nums);falseBoolean
3 String utils
 Is it empty or null:

String s = null;
U.isEmpty(s);trueBoolean

U.isEmpty("");trueBoolean

U.isEmpty("hey");falseBoolean

 Trim left by prefix:

U.triml("/abc/", "/");"abc/"String

 Trim right by suffix:

U.trimr("/abc/", "/");"/abc"String

 Substring supporting negative positions:

U.mid("abcde", 1, 2);"b"String // same as "abcde".substring(1, 2)

U.mid("abcde", 2, -1);"cd"String // -1 means "1 from right to left"

U.mid("abcde", 0, -2);"abc"String // -2 means "2 from right to left"
4 Dynamic implementation of interfaces
 Dynamic implementation of interfaces:

Dog dog = U.dynamic(Dog.class, (m, margs) -> m.getName() + ":" + margs[0]);

dog.bite("me");"bite:me"String

dog.eat("meat");"eat:meat"String
Dog.java
public interface Dog {

    String eat(String food);

    String bite(String target);

}
