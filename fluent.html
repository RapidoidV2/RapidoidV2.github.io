Rapidoid Fluent Fluent API for Clean Manipulation of Collections
Production-ready. ~ 80% test coverage. Self-contained. 31 KB JAR size.
Lightweight fluent DSL for elegant manipulation of collections and streams.


Add rapidoid-fluent as Maven dependency:
<dependency>
    <groupId>org.rapidoid</groupId>
    <artifactId>rapidoid-fluent</artifactId>
    <version>5.0.8</version>
</dependency>
1 Nice Fluent API with Java 8 streams behind the scenes
Using Java 8? Now you can EASILY filter, map, group and search through collections:
 
Find.firstOf(items).where(s -> s.length() < 3);
 
Find.allOf(nums).where((k, v) -> k > 1);
 
For.each(books).withNonNull(Book::title).run(this::publish);
 
Do.map(words).to(s -> s.length());
 
Do.group(items).by(Item::getCategory);
 
Do.reduce(words).by((a, b) -> a + ":" + b).orElse("");
 
Do.map(nums).toList((k, v) -> k * 2);
2 Why is Rapidoid Fluent so sweet?
Map<Integer, String> nums = New.map(1, "one", 2, "two", 3, "three");{"1":"one","2":"two","3":"three"}LinkedHashMap
 
Find the even numbers from the map - with Rapidoid Fluent (and Java 8 streams behind the scenes):
 
Find.allOf(nums).where((k, v) -> k % 2 == 0);{"2":"two"}HashMap
 
That is equivalent to the following code:
 
Map<Integer, String> even = nums.entrySet().stream()
        .filter(num -> num.getKey() % 2 == 0)
        .collect(Collectors.toMap(Entry::getKey, Entry::getValue));
 
System.out.println(even);{"2":"two"}HashMap
 
Or, we can do the same thing in the old way:
 
Map<Integer, String> even2 = new HashMap<>();
 
for (Entry<Integer, String> num : nums.entrySet()) {
    int k = num.getKey();
    if (k % 2 == 0) {
        even2.put(k, num.getValue());
    }
}
 
System.out.println(even2);{"2":"two"}HashMap
3 Map, reduce and group-by on collections
Map:
 
List<String> items = New.list("a", "bbbbb", "cc");["a","bbbbb","cc"]ArrayList
 
Do.map(items).to(s -> s.length());[1,5,2]ArrayList
 
Reduce:
 
Do.reduce(items).by((a, b) -> a + ":" + b).orElse("");"a:bbbbb:cc"String
 
Do.reduce(items).by("@", (a, b) -> a + ":" + b);"@:a:bbbbb:cc"String
 
Group-by:
 
Do.group(items).by(s -> s.length());{"1":["a"],"2":["cc"],"5":["bbbbb"]}HashMap
4 Map and group-by on maps
Map<Integer, String> nums = New.map(1, "one", 2, "two", 3, "three");{"1":"one","2":"two","3":"three"}LinkedHashMap
 
Map:
 
Do.map(nums).to((k, v) -> k * 1000, (k, v) -> v.toUpperCase());{"2000":"TWO","3000":"THREE","1000":"ONE"}HashMap
 
Do.map(nums).toList((k, v) -> k * 2);[2,4,6]ArrayList
 
Do.map(nums).toSet((k, v) -> v.length());[3,5]HashSet
 
Group-by:
 
Do.group(nums).by((k, v) -> k % 2 == 0 ? "even" : "odd");{"even":{"2":"two"},"odd":{"1":"one","3":"three"}}HashMap

6 Searching through lists or sets
Searching through lists or sets:
 
List<String> items = New.list("a", "bbbbb", "cc");["a","bbbbb","cc"]ArrayList
 
Find.firstOf(items).where(s -> s.length() < 3);"a"Optional<String>
 
Find.lastOf(items).where(s -> s.length() > 4);"bbbbb"Optional<String>
 
Find.anyOf(items).where(s -> s.length() == 5);"bbbbb"Optional<String>
 
Find.allOf(items).where(s -> s.length() > 1);["bbbbb","cc"]ArrayList
 
Find.in(items).where(s -> s.length() < 5);trueBoolean
 
Find.in(items.stream()).where(s -> s.length() > 10);falseBoolean
7 Searching through maps
Searching through maps:
 
Map<Integer, String> nums = New.map(1, "one", 2, "two", 3, "three");{"1":"one","2":"two","3":"three"}LinkedHashMap
 
Find.firstOf(nums).where((k, v) -> k >= 2);{"2":"two"}Optional<Entry>
 
Find.lastOf(nums).where((k, v) -> v.length() > 4);{"3":"three"}Optional<Entry>
 
Find.anyOf(nums).where((k, v) -> v.length() == 5);{"3":"three"}Optional<Entry>
 
Find.allOf(nums).where((k, v) -> k > 1);{"2":"two","3":"three"}HashMap
 
Find.in(nums).where((k, v) -> k == 2);trueBoolean
 
Find.in(nums).where((k, v) -> k > 10);falseBoolean
8 interface Flow extends Stream { with extra methods for convenience }
Flow == Java 8 Stream++:
 
Set<Character> chars = Flow.chars('a', 'e').toSet();["a","b","c","d","e"]HashSet
 
List<Long> nums = Flow.count(1, 7).toList();[1,2,3,4,5,6,7]ArrayList
 
List<Long> range = Flow.range(1, 7).toList();[1,2,3,4,5,6]ArrayList
 
Flow.chars('a', 'e').map(c -> c + "!").toList();["a!","b!","c!","d!","e!"]ArrayList
 
Flow.chars('a', 'd').map(c -> c + "!").map(String::toUpperCase).toList();["A!","B!","C!","D!"]ArrayList
 
Flow.chars('a', 'c').map(c -> c + "!").filter(s -> !s.startsWith("b")).toList();["a!","c!"]ArrayList
 
Flow.of("Hi", "there", "X").map(c -> c + "!").filter(s -> s.length() <= 3).toList();["Hi!","X!"]ArrayList
