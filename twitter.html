1 Rapidoid Coolness: RESTful Memory Info Service

Runtime rt = Runtime.getRuntime();
 
On.get("/memory").json(() -> U.map(
    "free", rt.freeMemory(),
    "total", rt.totalMemory(),
    "max", rt.maxMemory()
));
GET /memory
{"free":609009480,"total":803733504,"max":1828716544}

2 Super-simple RESTful JPA persistence with Rapidoid

@Entity
public class Movie {
    public String title;
}
 
// In the Main class:
On.get("/foo").html(req -> "bar");
 
On.get("/movies").html(() -> JPA.of(Movie.class).all());
GET /foo
bar
