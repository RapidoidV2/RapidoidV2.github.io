Rapidoid - Extremely Fast, Simple and Powerful Java Web Framework!

Open source (Apache license). View benchmark... Community: rapidoid@googlegroups.com

Rapidoid consists of several modules which can be used separately or together:
  • rapidoid-http-fast - Low-level HTTP Framework JDK 6 - 8
  • rapidoid-http-server - High-level Web Framework JDK 6 - 8
  • rapidoid-gui - Web GUI Components (based on Bootstrap + Angular) JDK 6 - 8
  • rapidoid-web - rapidoid-http-server + rapidoid-gui + goodies JDK 6 - 8
  • rapidoid-quick - rapidoid-web + Hibernate + Logback + MySQL Connector... JDK 6 - 8
  • rapidoid-render - Rapidoid's template engine JDK 6 - 8
  • rapidoid-fluent - Fluent API for Elegant Stream Manipulation JDK 8
  • rapidoid-essentials - Simple & Sweet Utils JDK 6 - 8
  • rapidoid-net - Network protocol framework JDK 6 - 8
Add rapidoid-essentials as Maven dependency:
<dependency>
    <groupId>org.rapidoid</groupId>
    <artifactId>rapidoid-essentials</artifactId>
    <version>5.2.6</version>
</dependency>
